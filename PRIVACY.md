# Privacy Policy — WhyTab

Last updated: April 30, 2025

## Data collected

WhyTab stores the following data locally on your device using chrome.storage.local:
- Tab titles, URLs, and favicons
- Opening timestamps and last-visited timestamps
- User-provided intent notes
- AI-inferred intent phrases

## Data transmitted

If you configure a Groq API key, WhyTab sends the title and URL of new tabs to the Groq API (api.groq.com) to infer why the tab was opened. No other data is transmitted to any external server.

Your Groq API key is stored locally in chrome.storage.local and is never transmitted anywhere other than directly to Groq's API.

## Data sharing

WhyTab does not share, sell, or transmit your data to any third party, except as described above (Groq, only if you opt in by providing an API key).

## Analytics

WhyTab contains no analytics, telemetry, or usage tracking of any kind.
